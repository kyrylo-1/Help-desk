   # Help-desk

### How to setup locally?
- go to _"help-desk-client_ folder" -> run **"npm i"** 

- go to _"Help-desk.API/Help-desk.API_ folder" -> run **"dotnet run"** 
- got to _"help-desk-client_ folder" -> run **"npm run start"**
